# Doneit

A TUI todo manager

# Install
`cargo install --path .`

# Keybindings
```
Tab         switch between todos and workspaces view
q | Esc     quit

j | k       navigate down | up
J | K       move item down | up
h | l       collapse | expand item
i           edit item
a           add sibling
A           add child
c           mark as completed

y           yank item 
x           cut item 
p           paste item as sibling
P           paste item as child 

+ | -       increase | decrease priority
```
